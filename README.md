# Reyaansh-Sketchware-Pro

A customized fork of Sketchware-Pro dedicated to opening up Java sources, exploring the codebase, and testing out local modifications. This repository serves as an open workspace for experimenting with new updates, UI tweaks, and build optimizations.

## 🌟 About This Fork

* **Accessible Codebase:** Making the Java sources easy to look through, modify, and experiment with.
* **Open Development:** A casual space to test out changes, fix random bugs, and try out new features without a strict roadmap.
* **Community Friendly:** Anyone is welcome to fork this, experiment with their own ideas, or submit a pull request if they find something cool to improve.

## 🛠️ Getting Started & Building

To clone the repository and look through the source code locally:

git clone https://github.com/reyaansh72/Reyaansh-Sketchware-Pro.git
cd Reyaansh-Sketchware-Pro

### Build Commands
To compile a local debug APK:

./gradlew assembleDebug

## 🤝 Contributions

Found a bug or want to test a new feature? Feel free to fork this repository, tinker with the code, and open a pull request with your updates!
